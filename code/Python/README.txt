octoDAC Python package

Install with 
> pip install -e . 

.\octoDAC\driver\octoDACDriver.py is main class file.
.\octoDAC\driver\octoDACdemo.py includes demo syntax for driver.

.\octoDAC\validation\octoDACTestScript.py includes tests for performance of device.  Includes examples of driving NicoLase and octoDAC on same serial port device.
.\octoDAC\validation\octoDACTestAnalysis.py generates metrics from data collected in octoDACTestScript.py
